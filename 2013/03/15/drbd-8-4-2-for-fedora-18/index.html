drbd 8.4.2 for Fedora 18

2013-03-15

If you use DRBD on Fedora 18, there’s a new client tools package on its way to the stable repositories. The kernel module for DRBD was bumped to 8.4.2 and the client tools have been updated to match it.

This fixes a bug that was submitted earlier this month.
